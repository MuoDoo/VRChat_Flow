# VRCFlow ユーザーガイド

[中文](USER_GUIDE_zh-CN.md) | [English](USER_GUIDE.md) | 日本語

VRCFlow は VRChat 向けのリアルタイム音声翻訳アプリです。マイクに話しかけると、音声が自動的に認識・翻訳され、VRChat の Chatbox に表示されます。

## ダウンロードとインストール（Windows）

1. [Releases ページ](https://github.com/MuoDoo/VRChat_Flow/releases) にアクセス
2. 最新の `.exe` インストーラーをダウンロード（例: `VRCFlow-Setup-0.0.5.exe`）
3. インストーラーを実行し、インストール先を選択してセットアップを完了
4. スタートメニューまたはデスクトップのショートカットから **VRCFlow** を起動

> インストーラーは Windows x64（Windows 10/11）用です。クライアントに追加のランタイムは不要です。

## はじめに

### 1. アカウント登録

VRCFlow を初めて開くと、ログイン画面が表示されます。

1. 画面下部の **「アカウントをお持ちでない方はこちら」** をクリック
2. ユーザー名（3〜20文字、英数字・アンダースコア）とパスワード（8文字以上）を入力
3. **登録** をクリック
4. *「登録が完了しました。管理者の承認をお待ちください。」* と表示されます

アカウントは管理者の承認後に利用可能になります。サーバー管理者に連絡して、有効化をお待ちください。

### 2. ログイン

アカウントが承認されたら：

1. ユーザー名とパスワードを入力
2. **ログイン** をクリック
3. メイン画面に移動します

### 3. 設定

右上の **設定** ボタンをクリックして設定パネルを開きます：

| 設定項目 | 説明 | デフォルト |
|----------|------|-----------|
| **サーバー URL** | VRCFlow サーバーのアドレス | `https://vrcflow.com` |
| **OSC ポート** | VRChat OSC の UDP ポート | `9000` |
| **原語** | あなたが話す言語 | 中国語 |
| **翻訳先言語** | 翻訳する言語 | 英語 |

選択可能な言語：中国語、英語、日本語、韓国語。

パネル内の **設定** ボタンで保存、**×** でキャンセルです。

### 4. UI 言語の切替

右上の言語ドロップダウンで、インターフェースを **English**、**中文**、**日本語** に切り替えられます。設定は自動的に保存されます。

## 使い方

### 翻訳を開始する

1. 画面下部の緑色の **開始** ボタンをクリック
2. システムからマイクの許可を求められた場合は許可してください
3. ステータスが **「リスニング中...」** と表示されたら、普通に話してください
4. 話し終えて間が空くと、VRCFlow が自動的に音声の終了を検出し、サーバーに送信して認識・翻訳します
5. 翻訳結果がメインエリアにカード形式で表示されます：
   - タイムスタンプ
   - 原文（話した内容）
   - 翻訳文（翻訳先言語のテキスト）
6. 翻訳テキストは同時に OSC 経由で VRChat の Chatbox に送信されます

### 翻訳を停止する

赤い **停止** ボタンをクリックしてリスニングを停止します。

### ステータス表示

| ステータス | 意味 |
|-----------|------|
| **待機中** | アイドル状態、「開始」で開始 |
| **リスニング中...** | マイクがアクティブ、音声入力待ち |
| **処理中...** | 音声を検出、サーバーに送信中 |
| 緑の丸 | リスニング中 |
| オレンジの丸 | 音声セグメントを処理中 |

### 1日の使用量クォータ

各アカウントには1日の使用時間制限があります（初回翻訳後、設定で残り秒数を確認できます）。残り時間は毎日リセットされます。

## VRChat の設定

VRChat の Chatbox に翻訳結果を表示するには：

1. VRChat を起動
2. **Action Menu** を開く（デスクトップ版は R キー）
3. **Options** → **OSC** → **Enable** を選択
4. VRCFlow の OSC ポートが **9000**（VRChat のデフォルト）に設定されていることを確認
5. 翻訳結果がアバターの上の Chatbox に表示されます

> VRChat と VRCFlow が同じ PC 上で動作している場合、追加設定は不要です。別のマシンで動作している場合は、設定で OSC のターゲット IP を調整してください。

## トラブルシューティング

### マイクが動作しない

- **Windows**：設定 → プライバシーとセキュリティ → マイク で、アプリのアクセスが有効になっていることを確認
- Windows のサウンド設定で、マイクがデフォルトの録音デバイスに設定されていることを確認
- マイクを使用している他のアプリを閉じてみてください

### 「アカウントが有効化されていません」と表示される

アカウントがまだ管理者に承認されていません。サーバー管理者にお問い合わせください。

### 「セッションが期限切れです。再度ログインしてください」と表示される

ログインセッションが期限切れになりました。ユーザー名とパスワードで再ログインしてください。

### 「1日の上限に達しました」と表示される

1日のクォータを使い切りました。制限は毎日深夜（サーバー時間）にリセットされます。翌日に再度お試しください。

### VRChat に翻訳結果が表示されない

- VRChat で OSC が有効になっていることを確認（Action Menu → Options → OSC → Enable）
- VRCFlow の OSC ポートが VRChat と一致していることを確認（デフォルト: 9000）
- 初めて OSC を有効にした場合は、VRChat を再起動してみてください

### 「Loading VAD...」と表示されたまま

音声検出モデルをロードしています。初回起動時は数秒かかります。長時間変わらない場合は、アプリを再起動してください。

### サーバーに接続できない

- インターネット接続を確認してください
- 設定のサーバー URL が正しいことを確認してください
- サーバーが一時的に利用できない可能性があります — しばらくしてから再試行してください
