# VRCFlow ユーザーガイド

[中文](USER_GUIDE_zh-CN.md) | [English](USER_GUIDE.md) | 日本語

VRCFlow は VRChat 向けのリアルタイム音声翻訳アプリです。マイクに話しかけると、音声が自動的に認識・翻訳され、VRChat の Chatbox に表示されます。

## ダウンロードとインストール（Windows）

1. [Releases ページ](https://github.com/MuoDoo/VRCFlow/releases) にアクセス
2. 最新の `.exe` インストーラーをダウンロード
3. インストーラーを実行し、インストール先を選択してセットアップを完了
4. スタートメニューまたはデスクトップのショートカットから **VRCFlow** を起動

> インストーラーは Windows x64（Windows 10/11）用です。追加のランタイムは不要です。

## はじめに

### 1. プロバイダーを選択する

VRCFlow は複数の ASR/翻訳プロバイダーをサポートしています。**設定**（右上）を開き、**Provider** タブで選択します。

#### Aliyun DashScope（百錬）

組み込みのオリジナルプロバイダー。Alibaba Cloud のリアルタイム WebSocket API を使用して、音声認識と翻訳を同時に行います。

**モデル：gummy-chat-v1**

| 特徴 | 詳細 |
|------|------|
| **プロトコル** | WebSocket デュプレックスストリーミング（リアルタイム） |
| **ASR + 翻訳** | 1回の呼び出しで認識と翻訳を同時実行 |
| **対応言語** | 中国語、英語、日本語、韓国語 |
| **料金** | ¥0.00015/秒/チャンネル、デュアル課金（ASR + 翻訳 = 2× 時間） |
| **無料枠** | 新規ユーザーに10時間の無料枠 |
| **課金方式** | 実際の発話時間のみ課金 |

**メリット：**
- 超低レイテンシー — リアルタイムストリーミング、話しながら結果が表示される
- VRChat の会話における短い音声セグメントに最適化
- シンプルな料金体系、トークン計算不要
- 中国語 ↔ 他言語の翻訳に高いコストパフォーマンス

**API Key の取得方法：**
1. [Alibaba Cloud 百錬 API Key ページ](https://bailian.console.aliyun.com/cn-beijing/?apiKey=1&tab=model#/api-key) にアクセス
2. アカウントがない場合は登録してください（新規ユーザーは10時間の無料枠あり）
3. API Key を作成してコピーします

#### OpenRouter

統一 API を通じて複数の AI モデルにルーティングする代替プロバイダー。柔軟なモデル選択と透明な価格設定を提供します。

**API Key の取得方法：**
1. [OpenRouter Keys](https://openrouter.ai/keys) にアクセス
2. アカウントを作成し、API Key を生成
3. アカウントにクレジットを追加

##### モデル：Voxtral Small 24B（推奨）

Mistral の音声対応モデル。最先端の音声文字起こしと翻訳機能を備えています。

| 特徴 | 詳細 |
|------|------|
| **プロバイダー** | Mistral AI（OpenRouter 経由） |
| **プロトコル** | REST API + Function Calling |
| **コンテキストウィンドウ** | 32K トークン |
| **最大出力** | 8,192 トークン |
| **音声入力料金** | $100 / 1M 秒（約 $0.006/分） |
| **テキスト入力料金** | $0.10 / 1M トークン |
| **テキスト出力料金** | $0.30 / 1M トークン |
| **音声フォーマット** | WAV、MP3、FLAC、Opus、OGG |

**メリット：**
- 業界最高水準の音声文字起こしと翻訳精度
- 短い音声セグメントのコストが非常に低い（VRChat に最適）
- 優れたテキスト理解性能を維持
- Function Calling 対応で構造化出力が信頼性の高い

##### モデル：Gemini 3.1 Flash Lite

Google の高効率モデル。大量処理・コスト重視のユースケースに最適化されています。

| 特徴 | 詳細 |
|------|------|
| **プロバイダー** | Google（OpenRouter 経由） |
| **プロトコル** | REST API + Function Calling |
| **コンテキストウィンドウ** | 1M トークン |
| **最大出力** | 65,536 トークン |
| **音声入力料金** | $0.50 / 1M トークン |
| **テキスト入力料金** | $0.25 / 1M トークン |
| **テキスト出力料金** | $1.50 / 1M トークン |
| **音声フォーマット** | WAV、MP3、FLAC、Opus、OGG |

**メリット：**
- 1M トークンの超大コンテキストウィンドウ
- テキスト、画像、動画、ファイル、音声入力に対応
- 暗黙的キャッシュ対応（キャッシュ読み取り：$0.025/1M トークン）
- Gemini 3 Flash の半額のコスト

### プロバイダー比較

| | DashScope | OpenRouter (Voxtral) | OpenRouter (Gemini) |
|---|---|---|---|
| **最適な用途** | 中国語 ↔ 他言語、最低レイテンシー | 最高精度、短い音声 | 大量処理、コスト重視 |
| **レイテンシー** | リアルタイムストリーミング | REST（ワンショット） | REST（ワンショット） |
| **料金モデル** | 秒単位 | 秒 + トークン | トークン |
| **1分あたり約** | ~¥0.018（約 $0.0025） | ~$0.006 | 音声の長さによる |
| **無料枠** | 10時間 | なし | なし |
| **モデル選択** | 固定（gummy-chat-v1） | 選択可能 | 選択可能 |

### 2. 設定

右上の **設定** ボタンをクリックして設定パネルを開きます：

**Provider タブ：**

| 設定項目 | 説明 |
|----------|------|
| **Provider** | DashScope または OpenRouter を選択 |
| **API Key** | 選択したプロバイダーの API キー |
| **モデル** | モデル選択（OpenRouter のみ） |

**一般タブ：**

| 設定項目 | 説明 | デフォルト |
|----------|------|-----------|
| **原語** | あなたが話す言語 | 中国語 |
| **翻訳先言語** | 翻訳する言語 | 英語 |
| **OSC ポート** | VRChat OSC の UDP ポート | `9000` |
| **SteamVR オーバーレイ** | VR オーバーレイ表示の有効/無効 | オフ |
| **処理タイムアウト** | API 応答を待つ最大秒数 | `5` |
| **スピーチパッド** | 発話終了検出前の無音パディング（ミリ秒） | `600` |
| **表示通貨** | 費用表示の通貨（CNY/USD/JPY） | CNY |

選択可能な言語：中国語、英語、日本語、韓国語。

### 3. UI 言語の切替

右上の言語ドロップダウンで、インターフェースを **English**、**中文**、**日本語** に切り替えられます。設定は自動的に保存されます。

## 使い方

### マイク翻訳

1. 画面下部の緑色の **開始** ボタンをクリック
2. システムからマイクの許可を求められた場合は許可してください
3. ステータスが **「聴取中...」** と表示されたら、普通に話してください
4. 話し終えて間が空くと、VRCFlow が自動的に音声の終了を検出し、認識・翻訳します
5. 翻訳結果がメインエリアにカード形式で表示されます：
   - タイムスタンプ
   - 原文（話した内容）
   - 翻訳文（翻訳先言語のテキスト）
   - プロバイダーと処理情報
6. 翻訳テキストは同時に OSC 経由で VRChat の Chatbox に送信されます

> 課金は発話中のみ発生します。「開始」を押しても、話さなければ料金はかかりません。

### スピーカー / システム音声翻訳

VRCFlow はシステム音声もキャプチャして翻訳できます（例：VRChat で他の人が話している内容）：

1. 下部の **スピーカー** タブをクリック
2. **開始** をクリックしてシステム音声のキャプチャを開始
3. アプリがデスクトップ音声をキャプチャし、VAD を適用して検出された音声を翻訳
4. 結果はマイク翻訳と一緒にスピーカーアイコン付きで表示

> この機能はボイスチャットで他の人が話している内容を翻訳するのに便利です。

### SteamVR オーバーレイ

設定 → 一般 → SteamVR オーバーレイで有効にすると、VR 内にフローティングパネルとして翻訳結果を表示できます：

- VR ビュー内に最近の翻訳をオーバーレイ表示
- メッセージは 8 秒後に自動消去
- 同時に最大 3 件のメッセージを表示

### 翻訳を停止する

赤い **停止** ボタンをクリックしてリスニングを停止します。

### ステータス表示

| ステータス | 意味 |
|-----------|------|
| **待機中** | アイドル状態、「開始」で開始 |
| **聴取中...** | マイク/スピーカーがアクティブ、音声入力待ち |
| **処理中...** | 音声を検出、認識・翻訳中 |
| 緑の丸 | リスニング中 |
| オレンジの丸 | 音声セグメントを処理中 |

### ダッシュボード

ヘッダーの棒グラフアイコンをクリックして使用量ダッシュボードを表示：
- プロバイダー別の費用内訳（DashScope は CNY、OpenRouter は USD で表示）
- 本日の音声秒数、課金秒数、推定費用
- ノイズ検出統計（咳、環境音など）
- 日別消費明細テーブル（プロバイダー別の詳細付き）
- 通貨変換対応（CNY / USD / JPY）

### 履歴

ヘッダーの時計アイコンをクリックして、タイムスタンプ、所要時間、プロバイダー情報、トークン使用量を含む過去のすべての翻訳を閲覧できます。

## VRChat の設定

VRChat の Chatbox に翻訳結果を表示するには：

1. VRChat を起動
2. **Action Menu** を開く（デスクトップ版は R キー）
3. **Options** → **OSC** → **Enable** を選択
4. VRCFlow の OSC ポートが **9000**（VRChat のデフォルト）に設定されていることを確認
5. 翻訳結果がアバターの上の Chatbox に表示されます

> VRChat と VRCFlow が同じ PC 上で動作している場合、追加設定は不要です。

## トラブルシューティング

### マイクが動作しない

- **Windows**：設定 → プライバシーとセキュリティ → マイク で、アプリのアクセスが有効になっていることを確認
- Windows のサウンド設定で、マイクがデフォルトの録音デバイスに設定されていることを確認
- マイクを使用している他のアプリを閉じてみてください

### VRChat に翻訳結果が表示されない

- VRChat で OSC が有効になっていることを確認（Action Menu → Options → OSC → Enable）
- VRCFlow の OSC ポートが VRChat と一致していることを確認（デフォルト: 9000）
- 初めて OSC を有効にした場合は、VRChat を再起動してみてください

### 「Loading VAD...」と表示されたまま

音声検出モデルをロードしています。初回起動時は数秒かかります。長時間変わらない場合は、アプリを再起動してください。

### 認識に失敗する

- 設定の API Key が正しいことを確認してください（選択したプロバイダーに対応していることを確認）
- インターネット接続を確認してください
- **DashScope**：API の枠を使い切っている可能性があります — [百錬コンソール](https://bailian.console.aliyun.com/) で確認してください
- **OpenRouter**：アカウントの残高を確認してください — [OpenRouter](https://openrouter.ai/credits) で確認してください

### 遅延が大きい・タイムアウトする

- 設定 → 一般 → 詳細で **スピーチパッド** の値を小さくしてみてください
- 別のプロバイダーやモデルに切り替えてみてください
- ネットワーク接続の安定性を確認してください
