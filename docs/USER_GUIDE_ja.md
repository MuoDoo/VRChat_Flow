# VRCFlow ユーザーガイド

[中文](USER_GUIDE_zh-CN.md) | [English](USER_GUIDE.md) | 日本語

VRCFlow は VRChat 向けのリアルタイム音声翻訳アプリです。マイクに話しかけると、音声が自動的に認識・翻訳され、VRChat の Chatbox に表示されます。

## ダウンロードとインストール（Windows）

1. [Releases ページ](https://github.com/MuoDoo/VRCFlow/releases) にアクセス
2. 最新の `.exe` インストーラーをダウンロード（例: `VRCFlow-Setup-0.0.6.exe`）
3. インストーラーを実行し、インストール先を選択してセットアップを完了
4. スタートメニューまたはデスクトップのショートカットから **VRCFlow** を起動

> インストーラーは Windows x64（Windows 10/11）用です。追加のランタイムは不要です。

## はじめに

### 1. DashScope API Key を取得する

VRCFlow は Alibaba Cloud の DashScope API を使用して音声認識と翻訳を行います。ご自身の API Key が必要です：

1. [Alibaba Cloud 百錬 API Key ページ](https://bailian.console.aliyun.com/cn-beijing/?apiKey=1&tab=model#/api-key) にアクセス
2. アカウントがない場合は登録してください（新規ユーザーは10時間の無料枠あり）
3. API Key を作成してコピーします

### 2. 設定

右上の **設定** ボタンをクリックして設定パネルを開きます：

| 設定項目 | 説明 | デフォルト |
|----------|------|-----------|
| **DashScope API Key** | 音声認識・翻訳用の API キー | （必須） |
| **OSC ポート** | VRChat OSC の UDP ポート | `9000` |
| **原語** | あなたが話す言語 | 中国語 |
| **翻訳先言語** | 翻訳する言語 | 英語 |

選択可能な言語：中国語、英語、日本語、韓国語。

### 3. UI 言語の切替

右上の言語ドロップダウンで、インターフェースを **English**、**中文**、**日本語** に切り替えられます。設定は自動的に保存されます。

## 使い方

### 翻訳を開始する

1. 画面下部の緑色の **開始** ボタンをクリック
2. システムからマイクの許可を求められた場合は許可してください
3. ステータスが **「聴取中...」** と表示されたら、普通に話してください
4. 話し終えて間が空くと、VRCFlow が自動的に音声の終了を検出し、認識・翻訳します
5. 翻訳結果がメインエリアにカード形式で表示されます：
   - タイムスタンプ
   - 原文（話した内容）
   - 翻訳文（翻訳先言語のテキスト）
6. 翻訳テキストは同時に OSC 経由で VRChat の Chatbox に送信されます

> 課金は発話中のみ発生します。「開始」を押しても、話さなければ料金はかかりません。

### 翻訳を停止する

赤い **停止** ボタンをクリックしてリスニングを停止します。

### ステータス表示

| ステータス | 意味 |
|-----------|------|
| **待機中** | アイドル状態、「開始」で開始 |
| **聴取中...** | マイクがアクティブ、音声入力待ち |
| **処理中...** | 音声を検出、認識・翻訳中 |
| 緑の丸 | リスニング中 |
| オレンジの丸 | 音声セグメントを処理中 |

### ダッシュボード

ヘッダーの棒グラフアイコンをクリックして使用量ダッシュボードを表示：
- 本日の音声秒数、課金秒数、推定費用
- ノイズ検出統計（咳、環境音など）
- 日別消費明細テーブル

### 履歴

ヘッダーの時計アイコンをクリックして、タイムスタンプと所要時間を含む過去のすべての翻訳を閲覧できます。

## VRChat の設定

VRChat の Chatbox に翻訳結果を表示するには：

1. VRChat を起動
2. **Action Menu** を開く（デスクトップ版は R キー）
3. **Options** → **OSC** → **Enable** を選択
4. VRCFlow の OSC ポートが **9000**（VRChat のデフォルト）に設定されていることを確認
5. 翻訳結果がアバターの上の Chatbox に表示されます

> VRChat と VRCFlow が同じ PC 上で動作している場合、追加設定は不要です。

## トラブルシューティング

### マイクが動作しない

- **Windows**：設定 → プライバシーとセキュリティ → マイク で、アプリのアクセスが有効になっていることを確認
- Windows のサウンド設定で、マイクがデフォルトの録音デバイスに設定されていることを確認
- マイクを使用している他のアプリを閉じてみてください

### VRChat に翻訳結果が表示されない

- VRChat で OSC が有効になっていることを確認（Action Menu → Options → OSC → Enable）
- VRCFlow の OSC ポートが VRChat と一致していることを確認（デフォルト: 9000）
- 初めて OSC を有効にした場合は、VRChat を再起動してみてください

### 「Loading VAD...」と表示されたまま

音声検出モデルをロードしています。初回起動時は数秒かかります。長時間変わらない場合は、アプリを再起動してください。

### 認識に失敗する

- 設定の DashScope API Key が正しいことを確認してください
- インターネット接続を確認してください
- API の枠を使い切っている可能性があります — [百錬コンソール](https://bailian.console.aliyun.com/) で確認してください
