# VRCFlow 使用指南

中文 | [English](USER_GUIDE.md) | [日本語](USER_GUIDE_ja.md)

VRCFlow 是一款 VRChat 实时语音翻译应用。对麦克风说话，语音会被自动识别、翻译，并显示在 VRChat 的 Chatbox 中。

## 下载安装（Windows）

1. 前往 [Releases 页面](https://github.com/MuoDoo/VRChat_Flow/releases)
2. 下载最新的 `.exe` 安装包（如 `VRCFlow-Setup-0.0.5.exe`）
3. 运行安装程序，选择安装路径，完成安装
4. 从开始菜单或桌面快捷方式启动 **VRCFlow**

> 安装包适用于 Windows x64（Windows 10/11），客户端无需额外运行环境。

## 快速上手

### 1. 注册账户

首次打开 VRCFlow 会看到登录界面。

1. 点击底部的 **"没有账户？注册"**
2. 输入用户名（3-20 个字符，支持字母、数字、下划线）和密码（至少 8 个字符）
3. 点击 **注册**
4. 提示注册成功，即可登录

### 2. 登录

1. 输入用户名和密码
2. 点击 **登录**
3. 进入主界面

### 3. 设置

点击右上角的 **设置** 按钮打开设置面板：

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| **服务器地址** | VRCFlow 服务器地址 | `https://vrcflow.com` |
| **OSC 端口** | VRChat OSC 的 UDP 端口 | `9000` |
| **源语言** | 你说的语言 | 中文 |
| **目标语言** | 翻译成的语言 | 英语 |

可选语言：中文、英语、日语、韩语。

点击面板内的 **设置** 按钮保存，或点 **×** 取消。

### 4. 切换界面语言

使用右上角的语言下拉菜单，可切换界面为 **English**、**中文** 或 **日本語**。切换后自动保存。

## 使用方法

### 开始翻译

1. 点击屏幕底部的绿色 **开始** 按钮
2. 系统弹出麦克风权限请求时，点击允许
3. 状态显示 **"正在聆听..."** — 正常说话即可
4. 停顿后，VRCFlow 自动检测到语音结束，发送到服务器进行识别和翻译
5. 翻译结果以卡片形式显示在主区域：
   - 时间戳
   - 原文（你说的内容）
   - 译文（翻译后的内容）
6. 译文会同时通过 OSC 发送到 VRChat 的 Chatbox

### 停止翻译

点击红色 **停止** 按钮停止监听。

### 状态指示

| 状态 | 含义 |
|------|------|
| **就绪** | 空闲状态，按「开始」启动 |
| **正在聆听...** | 麦克风已开启，等待语音输入 |
| **处理中...** | 检测到语音，正在发送到服务器翻译 |
| 绿色圆点 | 正在监听 |
| 橙色圆点 | 正在处理语音片段 |

### 每日使用额度

每个账户有每日使用时长限制（首次翻译后在设置中可查看剩余秒数），每天自动重置。

## VRChat 配置

在 VRChat 中显示翻译结果：

1. 打开 VRChat
2. 打开 **Action Menu**（桌面端按 R 键）
3. 进入 **Options** → **OSC** → **Enable**
4. 确认 VRCFlow 中 OSC 端口设置为 **9000**（VRChat 默认端口）
5. 翻译结果将显示在你头像上方的 Chatbox 中

> VRChat 和 VRCFlow 在同一台电脑上运行时无需额外配置。如果在不同设备上，需要在设置中调整 OSC 目标地址。

## 常见问题

### 麦克风无法使用

- **Windows**：进入 设置 → 隐私与安全 → 麦克风，确保已开启应用访问权限
- 在 Windows 声音设置中确认麦克风已设为默认录音设备
- 尝试关闭其他可能占用麦克风的程序

### 提示"会话过期，请重新登录"

登录已过期，使用账号密码重新登录即可。

### 提示"已达到每日上限"

你已用完当日额度，限制在每天零点（服务器时间）重置，明天再试。

### VRChat 中看不到翻译结果

- 确认 VRChat 已启用 OSC（Action Menu → Options → OSC → Enable）
- 确认 VRCFlow 的 OSC 端口与 VRChat 一致（默认 9000）
- 如果是首次启用 OSC，尝试重启 VRChat

### 显示"Loading VAD..."

语音检测模型正在加载，首次启动需几秒钟。如果长时间卡住，请重启应用。

### 无法连接服务器

- 检查网络连接
- 确认设置中的服务器地址正确
- 服务器可能暂时不可用，稍后重试
